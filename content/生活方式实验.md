# 生活方式实验

## 我的理想日常

> 规律、健康、有产出、有余裕。

## 具体时间表

我现在的每一天大概是这样：

| 时间 | 活动 |
|------|------|
| 早上 | 健身（力量训练或骑行） |
| 早上 | 做饭 |
| 中午 | 做饭、吃饭 |
| 下午 | 学习AI、探索新东西、了解信息 |
| 下午 | 买菜（偶尔） |
| 晚上 | 做饭、吃饭 |
| 晚上 | 刷抖音、玩手机游戏 |
| 晚上9-10点 | 睡觉 |

**规律但不高压。** 每天有事做，但不赶进度。

### 健身

健身不是"爱好"那么简单，它是我的**生活方式基础设施**。

- 力量训练，一周4-5次
- 骑行（穿插进行）
- 目标：减脂，保持低体脂

**每天早上锻炼 = 开机动作。**

训练完，人的状态被激活了，一天才有精神做其他事。这不是自律，是身体需要这个启动程序。

### 自己做饭

我每天自己做三餐。

- 成本比外卖低
- 可以控制食材和热量
- 做饭本身也是放松

### 收入

我想要的不是暴富，是**稳定且自主的收入**。

- 不需要月入几万
- 但需要有稳定的现金流
- 来源可以是：产品开发、内容收入、远程咨询

### 社交

我擅长和人交流，但不喜欢无意义的社交。

- 深度对话 > 浅层聚会
- 线上交流 + 偶尔的线下见面
- 在探索中认识人，而不是在社交场合硬撑

**现在在健身房会主动认识新朋友**，出去玩也会认识人。但前提是聊得来。

## 地点选择

### 当前：杭州富阳区

为什么还留在杭州：
- 基础设施好
- 技术氛围还在
- 租房成本低（1700）
- 一个人住，生活成本低

### 未来可能

- **低成本城市**：云南大理、成都周边、东南亚清迈
- **老家县城**：成本极低，但社交和技术资源少
- **数字游民路线**：在不同城市短住，体验后再决定
- **出国打工**：学好英语，去澳洲体验几个月（参见[[不上班的可能性]]）

## 支出水平

月均支出大概5000左右。

| 项目 | 金额 |
|------|------|
| 房租 | 1700 |
| 生活费（做饭+日常） | ~3000 |
| **合计** | **~5000** |

40万存款 ÷ 5000 = 80个月 = **6年半**。

这个成本让我有足够的时间去探索，不用焦虑。

## 核心原则

1. **身体是底线** —— 没有健康，什么都谈不上
2. **时间是资产** —— 怎么花时间是最大的投资
3. **产出定义价值** —— 不管做什么，要有可见的产出
4. **留有余裕** —— 不把自己逼到没有退路
5. **允许放松** —— 晚上刷抖音、玩游戏，不是浪费时间，是充电
